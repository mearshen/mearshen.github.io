  • Comparing Two Continuous Collision Detection Algorithms on CPU and GPUs
    Liu Yao, Zhang Xinyu
    International Conference on Virtual Reality and Visualization (co-located with ChinaVR), September 23-25, 2016, Hangzhou

    本文针对文森特过滤算法与基于几何分析的过滤算法，进行了等价性推导，在CPU和GPU上分别用两种算法进行了仿真实验，对于实验中的一些特殊情形进行探讨，并对基于几何分析的过滤算法做了一些改进。

    论文主页

  • A Fast Algebraic Non-Penetration Filter for Continuous Collision Detection using Vincent Theorem
    Zhang Xinyu, Liu Yao
    Journal of Graphical Models, 2015 (SCI影响因子1.23)

    本文提出了基于文森特定理的连续碰撞检测过滤算法，根据一次或两次迭代的文森特定理判断碰撞是否发生，显著提高了碰撞检测的效率。

    论文主页

  • An Algebraic Culling Algorithm for Continuous Collision Detection using Sturm Theorem
    Zhang Xinyu, Liu Yao
    IEEE International Conference on Mechatronics and Automation, 2~5 August, 2015, Beijing, China.

    本文提出了基于斯图姆定理的连续碰撞检测过滤算法，根据碰撞方程的斯图姆序列，得到两个边界值序列的符号变量值，最后根据斯图姆定理判断碰撞方程的解存在性，判定碰撞是否发生，有效过滤伪碰撞对。

    论文主页

  • A Simple Filtering Algorithm for Continuous Collision Detection using Taylor Models
    Zhang Xinyu, Liu Yao
    IEEE International Conference on Mechatronics and Automation, 2~5 August, 2015, Beijing, China.

    本文提出了基于泰勒模型的连续碰撞检测过滤算法，根据待检测对的位置得到检测方程，然后根据泰勒模型求出这个三次方程在指定范围的值域，最后根据根的存在性定理确定指定区间根的存在情况，将不存在根的三次方程过滤掉，表明该碰撞不会发生。本算法能有效过滤伪碰撞对，提高碰撞检测实时性。
